## Respostas

### 1. Alunos que não gostam de nenhuma disciplina

Pelo princípio da inclusão-exclusão:

\[
|P \cup B| = |P| + |B| - |P \cap B| = 35 + 28 - 12 = 51
\]

Como a turma possui 60 alunos:

\[
60 - 51 = 9
\]

**Resposta:** 9 alunos não gostam de nenhuma das duas disciplinas.

### 14. Sistema de numeração posicional

Um sistema de numeração posicional é caracterizado pelo fato de que o valor de cada algarismo depende da posição em que ele aparece no número. Assim, o mesmo algarismo pode representar valores diferentes conforme esteja nas unidades, dezenas, centenas, milhares etc.

**Resposta:** é um sistema em que a posição do algarismo altera seu valor.

### 15. Por que o sistema binário é ideal para circuitos digitais

O sistema binário é ideal para circuitos digitais porque utiliza apenas dois símbolos, 0 e 1, que podem ser associados a dois estados físicos simples, como desligado/ligado ou ausência/presença de tensão elétrica. Isso facilita a construção, a leitura e o processamento das informações pelos circuitos eletrônicos.

**Resposta:** porque representa diretamente os dois estados dos circuitos digitais.

### 16. Conceito de base de um sistema de numeração

A base de um sistema de numeração indica a quantidade de símbolos diferentes usados para representar números nesse sistema. Ela também determina os pesos das posições, que são potências da base.

**Resposta:** a base é a quantidade de símbolos disponíveis e a referência dos pesos posicionais.

### 17. Pertinência e inclusão em teoria de conjuntos

A relação de pertinência ocorre entre um elemento e um conjunto. Por exemplo, se o elemento \(a\) pertence ao conjunto \(A\), escreve-se \(a \in A\).

A relação de inclusão ocorre entre dois conjuntos. Por exemplo, se todos os elementos de \(A\) também pertencem a \(B\), então \(A\) está contido em \(B\), isto é, \(A \subseteq B\).

**Resposta:** pertinência relaciona elemento e conjunto; inclusão relaciona conjunto e conjunto.

### 18. Conjunto complementar

O conjunto complementar de \(A\), em relação a um conjunto universo \(U\), é formado por todos os elementos que pertencem a \(U\), mas não pertencem a \(A\).

\[
A^c = U - A
\]

**Resposta:** é o conjunto dos elementos do universo que não estão no conjunto considerado.

### 19. Classificação das afirmações

1. O sistema hexadecimal utiliza símbolos de 0 a 9 e A a F. **Verdadeira (V)**.
2. O sistema binário utiliza apenas dois símbolos. **Verdadeira (V)**.
3. O sistema octal possui nove símbolos. **Falsa (F)**, pois possui oito símbolos, de 0 a 7.
4. Em um sistema posicional, a posição altera o valor do algarismo. **Verdadeira (V)**.

**Resposta:** **V, V, F, V**.

### 20. Classificação das afirmações

1. O sistema decimal possui base 10. **Verdadeira (V)**.
2. O sistema binário é utilizado em computadores. **Verdadeira (V)**.
3. O ábaco é um instrumento de cálculo posicional. **Verdadeira (V)**.
4. O sistema romano é um sistema posicional. **Falsa (F)**, pois o valor dos símbolos romanos não depende de posição decimal como nos sistemas posicionais.

**Resposta:** **V, V, V, F**.
