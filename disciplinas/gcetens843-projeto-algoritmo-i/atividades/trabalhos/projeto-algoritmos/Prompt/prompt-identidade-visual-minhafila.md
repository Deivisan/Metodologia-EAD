# Prompt — Identidade Visual FilaClara (Estilo Infográfico)

> **Objetivo:** Gerar imagens em estilo infográfico limpo, profissional e fácil de comparar para o grupo votar na identidade visual do projeto.
> 
> **Ferramenta recomendada:** Notebook LM (ou qualquer gerador de imagem com bons prompts como Flux, Midjourney, Leonardo, GPT-4o, etc.)
>
> **Estilo geral desejado:** Infográfico moderno, clean, minimalista, alto contraste, fácil de ler, com bom espaçamento, tipografia clara, visual de "design system" ou moodboard.

---

## Prompt 1 — Foco só nas cores do Protocolo de Manchester (infográfico simples)

```text
Crie uma imagem em estilo infográfico moderno, limpo e profissional.

Mostre as 5 cores do Protocolo de Manchester para o app FilaClara em grandes círculos ou swatches bem organizados.

Cores exatas:
- Vermelho (Emergência): #C62828
- Laranja (Muito Urgente): #EF6C00
- Amarelo (Urgente): #F9A825
- Verde (Pouco Urgente): #2E7D32
- Azul (Não Urgente): #1565C0

Para cada cor, coloque:
- Nome completo (ex: "Emergência")
- Código HEX grande e bem legível abaixo

Título no topo em negrito e destacado: "FilaClara - Cores do Protocolo de Manchester - Opção 01"

Design: fundo branco ou cinza muito claro, estilo flat moderno, alto contraste, fácil de comparar lado a lado, visual profissional de apresentação ou design system. 
Alta legibilidade, bom espaçamento, sem elementos desnecessários. --ar 16:9
```

## Prompt 2 — Paleta Completa (Manchester + Interface do App)

```text
Crie uma imagem em estilo infográfico moderno, limpo e organizado mostrando a identidade visual completa do projeto FilaClara.

Divida claramente em duas seções:

**Seção superior (título grande):** "Protocolo de Manchester (Risco)"
- Mostre as 5 cores de risco em swatches grandes
- Nome da cor + código HEX bem legível

**Seção inferior (título grande):** "Paleta de Interface do App"
- Cor primária
- Cor secundária
- Fundo de telas
- Texto principal
- Fundo de cards
- Cor de borda/divisores

Use exatamente estas cores:

Manchester (Risco):
- Vermelho (Emergência): #C62828
- Laranja (Muito Urgente): #EF6C00
- Amarelo (Urgente): #F9A825
- Verde (Pouco Urgente): #2E7D32
- Azul (Não Urgente): #1565C0

Interface do App:
- Primária: #0D47A1
- Secundária: #455A64
- Fundo de telas: #F5F7FA
- Texto principal: #1A1A1A

Título geral no topo em destaque: "FilaClara - Identidade Visual Completa - Opção 01"

Estilo: infográfico profissional, design system, clean, minimalista, alto contraste, bom espaçamento, fácil de ler e comparar. Fundo branco ou cinza claro. --ar 16:9
```

## Prompt 3 — Com Exemplo de Aplicação (UI + Cores)

```text
Crie uma imagem em estilo infográfico que mostre a identidade visual do app FilaClara aplicada na prática.

Parte superior: as 5 cores do Protocolo de Manchester em grandes círculos/swatches com nome e HEX.

Parte inferior: um mockup simples e limpo de tela de celular (estilo iPhone 15) mostrando uma tela do app (ex: "Sua posição na fila"), usando as cores da identidade:
- Bolinhas ou barras de risco coloridas usando as cores do Manchester
- Botões principais na cor primária
- Fundo claro
- Texto legível

Título no topo: "FilaClara - Aplicação da Identidade Visual - Opção 01"

Estilo infográfico clean, moderno, profissional, bom contraste, fácil de entender como as cores ficam no produto real. --ar 16:9
```

## Prompt 4 — Grid de Comparação (quando tiver várias opções)

```text
Crie uma imagem em grid organizado (2x2 ou 1x3) mostrando 3 ou 4 opções diferentes de identidade visual lado a lado para o projeto FilaClara.

Cada célula deve mostrar de forma clara e organizada:
- Nome da opção (ex: "Opção 01 - Clássica")
- As 5 cores do Manchester em círculos grandes
- Cor primária do app
- Código HEX principais (bem legíveis)

Título no topo em destaque: "FilaClara - Comparação de Identidades Visuais"

Design muito organizado, espaçado, fácil de comparar visualmente as opções. Estilo profissional de apresentação ou moodboard. Fundo branco ou cinza claro. --ar 16:9
```

## Dicas de uso

- Substitua "Opção 01", "Opção 02" etc. conforme for gerando.
- Depois de gerar a imagem, salve com nome claro na pasta `imagens-identidade/`.
- Atualize o arquivo `Identidade-visual.md` apenas com a imagem + nome da opção (sem colar o prompt lá).
- Os prompts ficam aqui na pasta `Prompt/` para referência futura.
