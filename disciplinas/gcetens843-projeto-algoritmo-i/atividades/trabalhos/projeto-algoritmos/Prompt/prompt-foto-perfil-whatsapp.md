# Prompt — Foto de Perfil do Grupo (WhatsApp)

Modelo: GPT Image 2
Formato: Quadrado (1:1) — ideal para foto de perfil do WhatsApp
Estilo: Design grafico moderno, cores solidas, tipografia bold

---

## Descricao

Criar uma imagem quadrada no formato 1:1 para ser usada como foto de perfil de um grupo de WhatsApp universitario. Deve ter aparencia de design grafico profissional, nao uma foto real. Use fundo com gradiente ou cor solida vibrante, com textos e elementos graficos sobrepostos.

## Conteudo

Titulo principal em destaque (fonte bold, legivel mesmo em tamanho pequeno):

FILAS INTELIGENTES
+ ACESSIBILIDADE

Abaixo, em fonte menor:

GCETENS843 — Projeto Algoritmo I
UFRB 2026.1

No canto inferior, em fonte pequena:

Apresentacao: 29/06

## Elementos visuais sugeridos

- Um icone sutil de fila/pessoas ou de senha numerada, ou um icone de acessibilidade (cadeirante, simbolo de inclusao) integrado ao design
- Cores: azul escuro (#1a237e) como fundo ou gradiente para verde-agua ou teal (#00897b) — cores que remetem a saude, confianca e tecnologia
- Texto em branco ou amarelo claro para contraste
- Design limpo, sem poluicao visual, pensado para ser legivel em tamanho reduzi do (circular do WhatsApp)

## Instrucoes finais

- Nao usar fotos de pessoas
- Nao usar logos de marcas ou instituicoes
- Foco em tipografia e cores
- Formato quadrado exato (1:1)
- A imagem sera cortada em circulo pelo WhatsApp, entao manter os elementos centrados
