# Protótipo FilaClara — Open Designer

> Código importado do projeto **Web Prototype** do Open Designer.

## Arquivos

| Arquivo | Função |
|---|---|
| `index.html` | Protótipo principal no frame do iPhone 15, com fluxo clicável do paciente |
| `admin.html` | Painel demonstrativo da clínica para gestão da fila |

## Como testar

Abra o arquivo `index.html` no navegador e siga o fluxo:

1. começar;
2. onboarding;
3. retirar senha;
4. escolher clínica;
5. escolher tipo de atendimento;
6. acompanhar fila;
7. simular chamada próxima;
8. ver painel público;
9. testar acessibilidade;
10. abrir painel da clínica.

## Observação

Este protótipo é demonstrativo e usa dados fictícios do projeto **FilaClara**: senha `A-028`, Clínica Vida Mais, 120 pacientes cadastrados e redução simulada de 64% da fila presencial.
