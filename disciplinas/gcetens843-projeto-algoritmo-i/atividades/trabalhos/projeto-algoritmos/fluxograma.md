## Objetivo

Desenhar o fluxo lógico do sistema de Filas Inteligentes, desde a entrada do paciente até o atendimento.

## O que o fluxograma deve mostrar

- [ ] Entrada do paciente (chegada, cadastro, senha)
- [ ] Tipo de fila (comum, prioritária, por tipo de atendimento)
- [ ] Algoritmo de chamada (ordem, prioridade)
- [ ] Notificação ao paciente
- [ ] Painel de acompanhamento
- [ ] Histórico de atendimentos

## Ferramentas sugeridas
- draw.io / diagrams.net
- Lucidchart
- Figma (para manter identidade visual)
