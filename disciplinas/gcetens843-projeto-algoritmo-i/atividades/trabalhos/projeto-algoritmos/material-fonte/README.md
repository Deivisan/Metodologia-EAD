## Estudo rápido dos dois documentos (para o grupo)

**Complementaridade:**
- **Ausiane** traz o olhar prático do fluxo real nas unidades (o "buraco" entre guichê e triagem), o Protocolo de Manchester detalhado, e forte ênfase em acessibilidade desde o primeiro contato (TEA, sensorial, etc.) + integração técnica com sistemas do governo.

**Recomendação:** Leia os dois na íntegra. O `proposta.md` é o local onde estamos condensando e decidindo o que usar de cada um.
